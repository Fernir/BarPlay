# Chordplay

Публикация песен с аккордами для гитары.

## Быстрый старт

```bash
# Бэкенд
cd backend
npm install
npm run dev

# Фронтенд (новый терминал)
cd frontend
npm install
npm run dev
```

## Стек

- **Backend**: NestJS, Prisma, PostgreSQL, JWT
- **Frontend**: React, Vite, TailwindCSS, TypeScript
