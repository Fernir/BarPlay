# Chordplay API

NestJS + Prisma + PostgreSQL

## Установка

```bash
npm install
.env  # настройте DATABASE_URL="postgresql://admin@localhost:5432/music_chords?schema=public"
JWT_SECRET="super-secret-key-2024"
JWT_EXPIRES_IN="7d"
PORT=3001
npm run dev
```

## Swagger UI

http://localhost:3001/api-docs

## Тесты

npm run test
